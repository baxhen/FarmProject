# Spec 1 — Miranda do Douro Property

**Score**: 71/100 — **BUY with conditions**
**Price**: €103,000
**Size**: 5 hectares
**Key features**: 2 wells, unfinished 1980s building, hillside terrain

## Executive Summary

Good property for low-input grazing, drought-tolerant perennials, and storage crops. Building rights uncertain — requires PIP verification. At asking price, total investment ~€120,000 for basic viability. Excellent for sheep/goats, olives, almonds, herbs. Poor for large-scale vegetables due to rocky soil and water constraints.

## Top 3 Strengths
1. Two wells + good water potential (needs flow test verification)
2. Continental climate = exceptional for chill-hour fruits, storage crops, herbs
3. Cross-border proximity (Spain 5km) + underserved demographic niches

## Top 3 Challenges
1. Building legality uncertain — unfinished 1980s structure, solo rústico classification
2. Rocky shallow soil limits vegetable production
3. Summer drought requires irrigation infrastructure
