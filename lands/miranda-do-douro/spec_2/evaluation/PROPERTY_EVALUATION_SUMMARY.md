- Specific designation: Vinhas (COS2023 code 2.2.1.1) — traditional vineyard zone

**Protected Areas**:
- RAN: NOT covered (NÃO ABRANGIDO) ✅
- REN: NOT covered (NÃO ABRANGIDO) ✅
- Natura 2000: Check ICNF
- Parque Natural do Douro Internacional: Check proximity

**Building Rights**:
- Existing: Agricultural support building ("construção de apoio ao espaço agrícola")
- Building equipped as residential: kitchen + fireplace, sink connections, 2 rooms, bathroom, septic
- **Key issue**: Building was legally constructed as agricultural support BUT has residential infrastructure. This could be:
  a) Legal: infrastructure was added within agricultural support building allowances
  b) Illegal: residential conversion without proper licensing
  c) Regularizable: pre-existing condition that can be formalized
- Dwelling pathway: Habitação do agricultor — requires agricultural exploitation demonstration. Existing fruit trees (5yr) + PDM Vinhas designation = strong agricultural use case.
- Tourism pathway: Empreendimentos turísticos isolados subject to special conditions per PDM

**Building Regularization Strategy**:
1. Verify building registration on caderneta predial (is it registered? as what?)
2. If registered as agricultural: apply for habitação do agricultor status
3. Demonstrate agricultural exploitation: 5yr almond orchard + potential vineyard + vegetable production
4. Building already has required infrastructure (kitchen, bathroom, septic) — this is an asset, not a liability, IF properly regularized
5. PIP (Pedido de Informação Prévia) strongly recommended before purchase

**Legal Confidence**: Moderate. Classification is clear. Building pathway exists. Specific conversion legality needs municipal confirmation.
