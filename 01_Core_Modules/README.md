# Core Modules — Farm Project Knowledge Base

Universal, country-independent knowledge modules. Apply to any farm in any country. Referenced by the 8 specialized Claude skills.

## Modules

| Module | Contents | Status |
|--------|----------|--------|
| Universal_Principles/ | 5 principles extracted from property evaluations | ✅ |
| Medicinal_Herbs/ | Growing, processing, and profiting from medicinal plants | ✅ |
| Plant_Nursery/ | Propagating and selling seedlings, grafting, nursery management | ✅ |
| Storage_Systems/ | Root cellars, curing, food preservation | ⬜ (content in skills + reviews) |
| Value_Added_Processing/ | Equipment, margins, regulatory pathways | ⬜ (content in skills + reviews) |
| Market_Niche_Identification/ | Niche discovery, pricing strategy | ⬜ (content in skills + reviews) |
| Soil_Building_Strategies/ | Compost, cover crops, soil monitoring | ⬜ (content in skills + reviews) |

⬜ = Content exists in reviews/ and skills/ but not yet extracted to standalone module files.

## Usage

Each module is a standalone reference. Skills reference modules for detailed frameworks. Land analyses reference modules for enterprise evaluation criteria.
