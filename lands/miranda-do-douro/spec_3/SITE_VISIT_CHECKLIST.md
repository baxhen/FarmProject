## 3. Water Verification — MOST CRITICAL

**No water source mentioned in listing. This is the #1 unknown.**

- [ ] **Well**: Any existing well? Ask seller directly before visit. If none:
  - [ ] Well drilling viability: ask neighbors about water table depth in Sendim area
  - [ ] Drilling cost estimate from local company: €3,000-8,000 typical
  - [ ] Success rate in this specific location? (ask drillers, not seller)
- [ ] **Public water**: Is Sendim public water available at this property? Connection cost?
- [ ] **Surface water**: Any seasonal stream? Signs of water flow on property?
- [ ] **Rainwater**: 400-500mm rain. Roof catchment viable if building constructed.
- [ ] **Neighbor wells**: Ask 2-3 neighbors: depth, flow rate, ever run dry?

**If no water source confirmed**: Factor €8,000-12,000 into total cost. This is a €22k property that needs €10k in water before it's usable.

---

## 4. Soil Testing

- [ ] **South slope**: Test at top, middle, bottom of slope. Differences?
- [ ] **"Zona propicia para horta"**: Where is the garden area? pH, texture, depth.
- [ ] **Oak tree areas**: Soil under oaks (acidic from leaf litter) vs open areas (neutral)
- [ ] **Depth to bedrock**: Schist bedrock in Miranda. How deep is soil?
- [ ] **Drainage**: South slope = good drainage. Verify.

---

## 5. Existing Resources

### Stone Walls (PREMIUM ASSET — value may exceed land price)
- [ ] **Walk entire perimeter**: Are walls truly complete? Any collapsed sections?
- [ ] **Height**: >1.2m = livestock-proof. <1m = needs reinforcement.
- [ ] **Condition**: Loose stones? Mortar condition? (traditional dry-stone = no mortar)
- [ ] **Linear meters**: Measure or GPS track. Estimated: 480-520m for 1.43ha.
- [ ] **Replacement value**: €50-100/m × 500m = €25,000-50,000. Document for valuation.
- [ ] **Gate locations**: Where are openings? Functional gates?

### Oak + Pine Trees
- [ ] **Count oaks (carvalhos)**: Species? Holm oak? Portuguese oak? Size/age?
- [ ] **Count pines (pinhos)**: Species? Timber value? Pine nut potential?
- [ ] **Firewood quantity**: "Bastante lenha" — how many m³ estimated?
- [ ] **Mushroom potential**: Oak = shiitake, pine = saffron milk cap (Lactarius deliciosus)

### Other
- [ ] **Views**: "Vistas previligiadas" — document with photos. Tourism/workshop asset.
- [ ] **Existing paths/tracks**: Within property? Access routes?

## 9. Visit Logistics

**Best time**: Late summer (Aug-Sep) — driest period, assess water stress + fire risk. OR winter (Jan-Feb) — see frost patterns on south slope.

**Bring**: Soil kit, GPS, camera, measuring wheel (for stone walls), binoculars.

**Duration**: 3-4 hours. 1.43ha + stone wall inspection takes time.
