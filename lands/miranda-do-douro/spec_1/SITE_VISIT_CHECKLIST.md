## 2. Legal Documents to Request

**Before visit**:
- [ ] Caderneta predial (land registry certificate) — confirms ownership, area, classification
- [ ] Certidão permanente (permanent land registry certificate) — encumbrances, mortgages
- [ ] Planta de localização (location plan) from câmara

**During/after visit**:
- [ ] **PIP (Pedido de Informação Prévia)** — CRITICAL. Is farmer dwelling possible on this parcel?
- [ ] PDM consultation: exact RAN/REN boundaries overlapping property
- [ ] ICNF consultation: Natura 2000 or protected area overlap
- [ ] Building registration: is the 1980s structure on caderneta? As what?
- [ ] Well licensing: are wells registered with APA?
- [ ] Access road: is it public or servidão (easement)? Documented?

**Red flag if seller won't provide**: caderneta predial. Walk away.

## 3. Water Verification

### Well 1
- [ ] **Flow rate test**: Run pump for 1 hour. Measure output. Target: >1,000 L/hr for domestic+irrigation.
- [ ] **Depth**: How deep is well? (affects pump type + cost)
- [ ] **Water quality**: Collect sample. Test for: potability (coliforms), pH, hardness, iron, nitrates. Lab: €50-150.
- [ ] **Seasonal variation**: Ask neighbors — does well run dry in August-September?
- [ ] **Pump condition**: If pump exists, age? Working? Submersible or surface?
- [ ] **Well license**: Registered with APA? Legal to use?

### Well 2
- [ ] Same tests as Well 1
- [ ] Is it a different aquifer depth? (redundancy value)

### General Water
- [ ] **Rainwater harvesting potential**: Roof area on existing structure? m²?
- [ ] **Surface water**: Any seasonal streams? Evidence of water flow paths?
- [ ] **Neighbor wells**: Ask neighbors about water table depth, reliability
- [ ] **Irrigation storage**: Where could tanks/pond be placed?

---

## 4. Soil Testing

- [ ] **pH test** (field kit, €10-20): Sample 5-10 locations across property. Map variations.
- [ ] **Texture test** (jar test): % sand/silt/clay. Dig 20-30cm at 3-5 locations.
- [ ] **Depth to bedrock**: Dig or probe at multiple points. Where is rock? Map shallow vs deep areas.
- [ ] **Organic matter**: Dark color? Earthworm count? (dig 30×30×30cm, count worms. >10 = good)
- [ ] **Drainage**: Pour 1L water into hole. How fast does it drain? (<5 min = too fast, >30 min = poor)
- [ ] **Existing vegetation indicators**: What's already growing? Nettles = fertility. Broom = acidity. Brambles = moisture.

**Key question**: Can this soil support 1ha of vegetables, or is it grazing-only as suspected?

## 5. Existing Resources Inventory

### Building (1980s Unfinished)
- [ ] **Structural assessment**: Walls straight? Roof? Damp? Cracks? Bring builder if serious.
- [ ] **Dimensions**: m² footprint. How many rooms?
- [ ] **Materials**: Stone? Concrete block? Traditional schist?
- [ ] **Roof condition**: Tiles intact? Beams sound?
- [ ] **Floor**: Concrete slab or dirt?
- [ ] **Electrification status**: Any wiring? Connection nearby?
- [ ] **Photos**: Every room, every wall, roof from inside, all angles

### Trees & Vegetation
- [ ] **Count + identify all trees**: Species, approx age, health
- [ ] **Existing fruit trees**: Any? (listing didn't mention — verify)
- [ ] **Oak trees**: Holm? Cork? Quantity?
- [ ] **Scrub/undergrowth**: Species? Invasive? Grazeable?

### Fencing & Boundaries
- [ ] **Walk entire perimeter**: Mark GPS track. Compare to caderneta boundaries.
- [ ] **Fencing condition**: Type? Intact? Gaps?
- [ ] **Boundary markers**: Stone posts? Paint marks? Natural features?
- [ ] **Encroachments**: Neighbor fences, buildings, crops crossing boundary?

### Terrain
- [ ] **Slope map**: Walk the land. Where is steepest? Flattest?
- [ ] **Aspect map**: Which directions do slopes face?
- [ ] **Erosion evidence**: Gullies, exposed rock, landslide scars?
- [ ] **Vehicle access**: Can a tractor reach all usable areas? Where not?

## 6. Infrastructure Verification

- [ ] **Electricity**: Nearest pole? Distance in meters. Get connection quote from E-Redes.
- [ ] **Road access**: Condition in wet weather? (Visit after rain if possible). Who maintains?
- [ ] **Phone/internet**: Check signal on phone. Ask neighbors what they use.
- [ ] **Waste**: Septic possible? (soil permeability test). Municipal waste collection?
- [ ] **Fire risk**: Defensible space around building? Water source for firefighting?

## 7. Neighbor & Community Interview

- [ ] **Water table**: How deep are their wells? Ever run dry?
- [ ] **Frost dates**: When is last spring frost? First fall frost? (local knowledge > climate data)
- [ ] **Wind**: Prevailing direction? Strength? Seasonal patterns?
- [ ] **Wildlife**: Wild boar? Deer? Rabbits? Predators?
- [ ] **Theft/vandalism**: Any issues with remote properties?
- [ ] **Community**: Is there an agricultural cooperative? Farmers market? Local events?
- [ ] **History**: What was this land used for previously? Why is seller selling?

## 8. Red Flags Specific to Spec 1

| Red Flag | Severity | What to Check |
|----------|----------|---------------|
| Building unregistered since 1980s | **HIGH** | If not on caderneta, regularization may be impossible. Could be treated as illegal construction. |
| RAN/REN status unknown | **HIGH** | If >30% of property is RAN/REN, building envelope severely restricted. Verify before offer. |
| Well flow never tested | **HIGH** | No flow data = can't confirm water security. Test during visit. |
| Remote location | MEDIUM | Confirm access road is legal right-of-way, not neighbor's goodwill. |
| 1980s materials | MEDIUM | Asbestos? Lead paint? Old construction materials can be hazardous. |

## 9. Visit Logistics

**Best time to visit**: Late August — see property at driest (water stress visible), hottest, and highest fire risk.

**Bring**:
- GPS phone (track walk, mark features)
- Soil test kit (pH + jars for texture)
- Measuring tape (30m)
- Camera (photograph everything)
- Notebook + property map
- Water bottles (collect well samples)
- Binoculars (assess distant parts of 5ha)
- Machete/secateurs (scrub may be thick)

**Duration**: Full day. 5ha takes time to walk properly.

**After visit**: Send water samples to lab. Request PIP from câmara. Get electricity connection quote. Calculate total development cost with real numbers.
